## 🚀 Déploiement

### Option 1 : Cloudflare Pages (Recommandé)

1. Connectez-vous à [Cloudflare Pages](https://pages.cloudflare.com/)
2. Créez un nouveau projet
3. Connectez votre dépôt GitHub
4. Commande de build : `npm run build` (si besoin)
5. Dossier de publication : `/` (racine)
6. Déployer !

### Option 2 : Vercel

1. Connectez-vous à [Vercel](https://vercel.com/)
2. Importez votre dépôt GitHub
3. Laissez les paramètres par défaut
4. Déployer !

### Option 3 : Netlify

1. Connectez-vous à [Netlify](https://netlify.com/)
2. Glissez-déposez le dossier du projet
3. Ou connectez votre dépôt GitHub
4. Déployer !

## 🔧 Configuration API

### Gemini API (Google)

1. Obtenez une clé API sur [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Modifiez `config-api.js` :
```javascript
const GEMINI_API_KEY = 'VOTRE_CLE_GEMINI';
```

### Groq API (Llama 3)

1. Obtenez une clé sur [Groq Console](https://console.groq.com/)
2. Modifiez `config-api.js` :
```javascript
const GROQ_API_KEY = 'VOTRE_CLE_GROQ';
```
